# PconCalendarApp

## Steps to run it locally

### Creating Virtual environment

```python3 -m venv PconFlaskvenv```

```cd PconCalendarApp```

### Activating the virtual environment

```source ../PconFlaskvenv/bin/activate```

```pip3 install wheel```

``` pip3 install -r requirements.txt```

### Running the app
```python3 run.py```
